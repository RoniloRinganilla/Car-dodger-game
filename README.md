# Python Mini Game with Lyrics 🎮🎶

A simple Python mini game made with **Tkinter**.  
You control a small block (the player) and avoid falling obstacles.  
When the player collides with an obstacle (game over), a set of song lyrics will automatically type out on the screen.

---

## Features ✨
- **Tkinter GUI** with a canvas-based game environment.
- Player movement using **Left** and **Right arrow keys**.
- Randomly spawning falling obstacles.
- **Game Over event** triggers an animated lyrics typing effect.
- Small watermark `@Arjinomoto` displayed at the top-left corner.

---

## Requirements 📦
- Python **3.8+**
- Tkinter (usually comes pre-installed with Python)
